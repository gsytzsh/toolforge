About Unicode Reference

This Unicode reference shows code points in U+hex and decimal with their characters. Useful for symbol lookups, copy-paste and encoding reference.

How to Use Unicode Reference

  1. Step 1: enter a character or code point to look up
  2. Step 2: Switch between hex and decimal representations as needed.
  3. Step 3: Resolve character details and encoding information.
  4. Step 4: Use output in text processing, escaping or debugging workflows.

Frequently Asked Questions

What is Unicode Reference mainly used for?
Unicode Reference is mainly used to inspect Unicode code points and character metadata quickly.
Does Unicode Reference send my input to a server?
For normal usage, Unicode Reference processes data locally in your browser so you can test quickly without uploading project files.
Why do Unicode characters display differently across apps?
Font support, normalization and rendering engines vary, which can change appearance for the same code point.
