About IP Subnet Calculator

This IP subnet calculator computes network address, broadcast address, CIDR notation and usable host range from an IP address and subnet mask (or CIDR prefix).

It is useful for network planning, subnet design, firewall rules and understanding IP addressing for both IPv4 and basic calculations.

How to Use IP Subnet Calculator

  1. Step 1: enter IP address and CIDR prefix
  2. Step 2: Confirm whether you are working in private or public address space.
  3. Step 3: Calculate network, broadcast and host ranges.
  4. Step 4: Use results for network planning, firewall rules or documentation.

Frequently Asked Questions

What is IP Subnet Calculator mainly used for?
IP Subnet Calculator is mainly used to calculate subnet ranges, mask size and usable host counts.
Does IP Subnet Calculator send my input to a server?
For normal usage, IP Subnet Calculator processes data locally in your browser so you can test quickly without uploading project files.
Why is my usable host count lower than expected?
Traditional IPv4 subnetting reserves network and broadcast addresses, reducing assignable host count.
