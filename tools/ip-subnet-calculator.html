About IP Subnet Calculator

This IP subnet calculator computes network address, broadcast address, CIDR notation and usable host range from an IP address and subnet mask (or CIDR prefix).

It is useful for network planning, subnet design, firewall rules and understanding IP addressing for both IPv4 and basic calculations.
