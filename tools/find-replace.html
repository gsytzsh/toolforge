About Find and Replace

This find and replace tool searches for text patterns and replaces them with your specified value. It supports both literal string matching and regular expressions (regex) for advanced pattern-based substitution.

Basic vs Regex Mode

Feature Basic Mode Regex Mode
Matching Literal text Patterns
Wildcards Not supported .*, \d, \w, etc.
Capture Groups No Yes ($1, $2...)
Character Classes No [A-Z], [0-9], etc.
Best For Simple replacements Complex patterns

Common Regex Patterns

Pattern Matches Example Match
\d+ One or more digits 123, 42, 0
\w+ Word characters hello, user_123
[A-Z] Uppercase letters A, B, C
\s+ Whitespace " ", "\t", "\n"
.* Any characters (greedy) anything
^word Starts with "word" word at line start
word$ Ends with "word" word at line end

Replacement Examples

Example 1: Remove all numbers
Find: \d+
Replace: (empty)
Input: "I have 123 apples and 456 oranges"
Output: "I have  apples and  oranges"

Example 2: Swap first and last name
Find: (\w+), (\w+)
Replace: $2 $1
Input: "Doe, John"
Output: "John Doe"

Example 3: Replace multiple spaces with single
Find: \s+
Replace: " "
Input: "too    many   spaces"
Output: "too many spaces"

Example 4: Wrap numbers in brackets
Find: (\d+)
Replace: [$1]
Input: "I have 42 dollars"
Output: "I have [42] dollars"

Special Characters in Basic Mode

In basic mode, these characters are treated literally:

  • . dot
  • * asterisk
  • + plus
  • ? question mark
  • [ ] brackets
  • ( ) parentheses
  • ^ $ caret and dollar
  • \ backslash

In regex mode, these have special meaning for pattern matching.

Using Capture Groups

Capture groups let you reuse matched text in replacements:

  1. Wrap part of pattern in parentheses: (pattern)
  2. Reference in replacement: $1, $2, etc.
  3. $1 = first group, $2 = second group
Example: Convert date format from MM/DD/YYYY to YYYY-MM-DD

Find: (\d{2})/(\d{2})/(\d{4})
Replace: $3-$1-$2

Input: "12/25/2024"
Output: "2024-12-25"

How to Use Find and Replace

  1. Enter find text: Type the text or regex pattern to search for.
  2. Enter replacement: Type what should replace found matches.
  3. Enable regex (optional): Check the box if using regex patterns.
  4. Paste input text: Enter the text to process.
  5. Click Replace All: All occurrences are replaced in the output.
  6. Copy result: Use the output in your documents or code.

Tips

  • Test regex patterns on small text first
  • Use capture groups for complex rearrangements
  • Empty replacement deletes all matches
  • Special characters need escaping in basic mode

Frequently Asked Questions

How does find and replace work?
Find and replace searches for a specific string or pattern in text and replaces all occurrences with a new value. In regex mode, you can use patterns like \d+ to match numbers or [A-Z] for uppercase letters.
What is regex mode and when should I use it?
Regex (regular expression) mode enables pattern-based matching instead of literal text. Use it for complex searches like 'all numbers', 'email addresses', or 'words starting with capital'. Enable the Regex checkbox to use it.
How do I replace all occurrences?
Both 'Replace' and 'Replace all' buttons replace all occurrences globally. Enter your find text and replacement text, then click either button. The result appears in the output box below.
Can I use capture groups in replacements?
Yes, in regex mode you can use capture groups. Use $1, $2, etc. in the replacement to reference captured groups. Example: Find '(\w+), (\w+)' Replace '$2 $1' converts 'Doe, John' to 'John Doe'.
Is the replacement case-sensitive?
Yes, by default matching is case-sensitive. 'hello' won't match 'Hello' or 'HELLO'. In regex mode, add the 'i' flag pattern or use character classes like [Hh]ello for case-insensitive matching.
