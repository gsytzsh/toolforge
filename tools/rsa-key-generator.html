About RSA Key Generator

This tool generates RSA-OAEP 2048-bit key pairs in the browser. Includes a demo to encrypt with the public key and decrypt with the private key.

Useful for learning RSA, testing key exchange and small message encryption. Note: RSA has a small payload limit (~190 bytes for 2048-bit).
