About IP ⇄ Number Converter

This converter transforms IPv4 addresses to 32-bit decimal numbers and decimal numbers back to IP. Useful for database storage, range comparisons and IP arithmetic.

It is useful when storing IPs as integers, comparing IP ranges, or converting between formats for APIs and databases.

How to Use IP Number Converter

  1. Step 1: paste an IPv4 value or numeric representation
  2. Step 2: Select conversion direction before running.
  3. Step 3: Convert and verify output format.
  4. Step 4: Copy converted value for databases, logs or scripts.
