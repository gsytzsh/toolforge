About Add Prefix and Suffix

This tool adds a prefix and/or suffix to each line of text. Useful for bullet lists, SQL values, config entries and more.

It is useful for formatting line lists into markdown, code, or structured data without manual editing.

How to Use Add Prefix Suffix

  3. Step 3: Click the action button to apply the changes.
  4. Step 4: Copy the processed text for your use.
