    About Password Strength Checker

    This password strength checker scores a password based on length, character variety and common weak patterns, then gives simple suggestions to improve it.

    It is useful when creating new passwords, reviewing internal password rules or checking whether a draft password is too short, too simple or too predictable before you use it.
