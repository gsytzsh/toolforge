Git Commands Reference

Quick reference for common Git commands. Init, clone, branch, commit, push, pull and more.

Initialize & Clone

CommandDescription
git initCreate a new Git repo in current directory
git init [project-name]Create directory and init as Git repo
git clone <url>Clone a remote repository

Config

CommandDescription
git config --listShow current Git config
git config [--global] user.name "name"Set author name
git config [--global] user.email "email"Set author email

Branch

CommandDescription
git branchList local branches
git branch -rList remote branches
git branch -aList all branches
git branch [name]Create branch (stay on current)
git checkout -b [branch]Create and switch to branch
git checkout [branch]Switch to branch
git merge [branch]Merge branch into current
git branch -d [branch]Delete branch

Commit & Status

CommandDescription
git statusShow working tree status
git add [file]Stage file(s)
git add .Stage all changes
git commit -m "message"Commit with message
git commit -am "message"Add tracked files and commit
git logShow commit history
git diffShow unstaged changes

Remote & Push / Pull

CommandDescription
git remote -vList remotes
git remote add origin <url>Add remote
git push [remote] [branch]Push to remote
git pull [remote] [branch]Pull and merge from remote
git fetchFetch from remote (no merge)

About Git Commands Reference

This Git commands reference lists common commands for initializing repos, branching, committing, and syncing with remotes.

It is useful when you need a quick reminder of syntax, setting up a new project, or onboarding to a Git-based workflow.
