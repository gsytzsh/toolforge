About ASCII Table

This ASCII table lists standard ASCII character codes from 0 to 127 in decimal, hex and character form.

It is useful for encoding references, control-character lookups and low-level text debugging when you need to identify exact character values.

What is ASCII Table mainly used for?

Does ASCII Table include extended non-ASCII characters?
