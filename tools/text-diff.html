About Text Diff


This text diff tool compares two text blocks line by line and highlights lines that appear only in one side.


It is useful for draft reviews, config comparisons, copy edits and quick change checks when you need a simple visual diff in the browser.

Frequently Asked Questions




  
What is Text Diff mainly used for?

  
Text Diff is mainly used to paste two text blocks to compare them line by line. Lines only in A or only in B are highlighted.


  
Does Text Diff send my input to a server?

  
For normal usage, Text Diff processes data locally in your browser so you can work without uploading project files.


  
Why does processed text differ from expectation?

  
Whitespace rules, hidden characters and line-ending differences are common reasons for unexpected text output.
