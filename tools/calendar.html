关于日历工具

这是一个轻量级的在线日历工具，可浏览任意月份、快速定位今天、并为特定日期添加备注。所有备注使用 localStorage 存储在您的浏览器本地，不会上传到任何服务器。

日历界面说明

元素 说明
月份标题 显示当前浏览的年月（如"January 2026"）
时区信息 显示浏览器所在时区（如"Asia/Shanghai"）
星期标题 Sun, Mon, Tue, Wed, Thu, Fri, Sat
日期格子 显示日期数字，可点击选中
蓝色边框 标记今天是哪一天
深色边框 标记当前选中的日期
橙色圆点 表示该日期有备注

功能按钮说明

按钮 功能
← Prev 切换到上一个月
Today 快速返回今天的日期
Next → 切换到下一个月
Save 保存当前选中日期的备注
Clear 清空当前日期的备注

localStorage 存储机制

备注使用 HTML5 localStorage API 存储在浏览器本地：

存储键名格式：toolforge.calendar.note:YYYY-MM-DD

示例：
键名：toolforge.calendar.note:2026-03-24
值：下午 3 点开会

存储特点：
- 数据永久保存，除非手动清除
- 每个域名约 5-10MB 空间
- 不会随浏览器关闭而丢失
- 不同浏览器/设备不共享

常见应用场景

  • 个人计划：记录每日待办事项、约会时间
  • 重要日期：标记生日、纪念日、截止日期
  • 工作安排：记录会议、项目里程碑
  • 学习笔记：为学习日期添加复习提醒
  • 轻量规划：不需要复杂功能的简单日历

日期格式说明

日历使用 ISO 8601 日期格式（YYYY-MM-DD）：

  • ISO 格式：2026-03-24（年 - 月 - 日）
  • 优点：全球统一标准，排序方便，无歧义
  • 存储键名：使用前缀 + ISO 日期确保唯一性

数据隐私说明

  • 本地存储：所有备注仅存储在您的浏览器中
  • 无服务器传输：数据不会发送到任何服务器
  • 无追踪：不收集任何个人信息或使用数据
  • 清除数据：可通过浏览器设置清除所有备注
  • 离线使用：页面加载后可离线使用

使用限制

  • 存储空间限制：每个域名约 5-10MB
  • 数据不跨设备同步（需手动导出导入）
  • 清除浏览器数据会删除所有备注
  • 仅支持文本备注，不支持附件

如何使用日历工具

  1. 浏览月份：使用"Prev"和"Next"按钮切换月份，或点击"Today"返回今天。
  2. 选中日期：点击任意日期格子选中该日期。
  3. 添加备注：在下方输入框中输入备注内容，点击"Save"保存。
  4. 查看备注：点击有橙色圆点的日期可查看或编辑备注。
  5. 清空备注：点击"Clear"按钮可删除当前日期的备注。

提示

  • 备注会自动保存在浏览器中，关闭页面后数据不丢失
  • 橙色圆点标记表示该日期已有备注
  • 蓝色边框标记今天，深色边框标记当前选中日期
  • 清除浏览器缓存会删除所有备注数据
  • 不同浏览器之间的数据不互通

常见问题解答

这个日历工具有什么特点？
这是一个轻量级在线日历，支持浏览任意月份、快速定位今天、为特定日期添加备注。所有备注使用 localStorage 存储在浏览器本地，不会上传到服务器，关闭浏览器后数据依然保留。
如何在日历中添加日期备注？
点击任意日期格子选中该日期，然后在下方的备注输入框中输入内容，点击"保存"按钮。有备注的日期右上角会显示橙色圆点标记，再次点击可查看或编辑备注。
备注数据存储在哪里？会同步吗？
备注使用 HTML5 localStorage 存储在您的浏览器本地。数据不会上传到任何服务器，因此不会在不同设备间同步。清除浏览器数据会删除备注。适合个人轻量级使用。
日历如何显示今天和选中的日期？
今天的日期用蓝色边框和阴影高亮显示。当前选中的日期用深色边框标记。点击"Today"按钮可快速返回今天。月份标题显示当前浏览的年月和时区信息。
日历的显示范围是什么？
日历网格显示 6 行×7 列共 42 天，确保完整覆盖任何月份（包括前后月份的补位日期）。星期标题显示英文缩写（Sun-Sat）。日期格子最小高度 72px，可容纳备注标记。
如何使用 localStorage 存储日期备注？
备注使用键值对存储，键名格式为"toolforge.calendar.note:YYYY-MM-DD"。ISO 日期格式确保唯一性。存储限制约 5-10MB，足够存储大量日期备注。使用 localStorage API 读写数据。
