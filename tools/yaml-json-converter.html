About YAML ⇄ JSON Converter

This YAML and JSON converter helps you switch between the two formats for configs, pipeline files and structured app data, with all parsing handled in the browser.

It is useful for Docker, CI, Kubernetes and app configuration work when you need to rewrite structured data without changing the actual values.
