About Extended ASCII Table

This Extended ASCII table lists character codes from 128 to 255 in decimal, hex and character form. Values 128–255 vary by encoding (Latin-1, Windows-1252, etc.).

It is useful for encoding references, special character lookups and debugging extended character display issues.
